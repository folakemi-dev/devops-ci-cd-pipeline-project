# DevOps CI/CD Pipeline Project

This project demonstrates a complete CI/CD workflow using:

- Terraform
- AWS EC2
- Jenkins
- SonarQube
- Nexus Repository
- Apache Tomcat
- GitHub

## Planned Architecture

GitHub -> Jenkins -> SonarQube -> Nexus -> Tomcat

## Project Goal

To provision infrastructure with Terraform, build a Java web application, analyze code quality with SonarQube, store artifacts in Nexus, and deploy automatically to Tomcat using Jenkins.
